В этой папке содержатся файлы с "сырыми" контекстами, полученными в результате парсинга корпуса BCCWJ.
Файлы получены в результате выполнения кода в тетрадках, содержащих в названии "corpus_building" (таких тетрадей две: одна для парсинга из VSC, одна - из Colab).
Файлы распределены по папкам по каждому из рассмотренных типов составных единиц.
